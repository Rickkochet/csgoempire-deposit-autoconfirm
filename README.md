# csgoempire-deposit-autoconfirm

Create a config file like:
```
{
    "steam": true, 
    "discord": true,
    "port": 3000,
    "domain": "localhost",
    "identitySecret": "your-identity-secret",
    "accountName": "your-steam-username",
    "password": "your-steam-password",
    "sharedSecret": "your-shared-secret",
    "mainCookie": "PHPSESSID=phpsessid-cookie; do_not_share_this_with_anyone_not_even_staff=donotsharethiswithanyonenotevenstaff_cookie",
    "discordHook": "discord webhook url",
    "discordUserId": "your discord user id to tag you when using the discord webhook"
}
```
